Recent Projects

Nothing to Wear

A web application where users can enter their zip code, plans for the day and dress code and receive several outfit suggestions. Nothing to Wear was built using Node.js, Express and PostgreSQL. In progress.

Blocipedia

An application that allows users to create, collaborate and share wikis. It was created using Node.js, Express and PostgreSQL. I integrated Stripe to charge users for premium accounts and implemented email sending and user authentication using external APIs.

Bloccit

A Reddit-like application created using Node.js. This web application has many features such as authentication, authorization and the ability for users to create topics and post comments. Bloccit incorporated PostgreSQL, Express, Jasmine and Passport, among others.

Bloc Chat

Bloc Chat was created using React, Firebase, HTML, CSS and JavaScript. The application allows users to sign in using their Gmail account, create new chatrooms and send messages in different chat rooms.

PA Giving Day Landing Page

This page was created as a landing page while waiting for a campaign to launch. Using a Bootstrap template and JavaScript, the page allowed users to view information about the campaign and see a countdown clock to generate excitement.

Bloc Jams

Bloc Jams is a music streaming application built using React, HTML, CSS and JavaScript. Users are able to pick an artist and browse songs on each album.
